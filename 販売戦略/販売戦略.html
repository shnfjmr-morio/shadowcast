チャネルUTMパラメータ

YouTubeショート?utm_source=youtube&utm_medium=video&utm_campaign=shorts

Product Hunt?utm_source=producthunt&utm_medium=listing&utm_campaign=launch
アフィリエイト?utm_source=affiliate&utm_campaign=aff-{id}
紹介リンク?ref={user_id}

実装コード（index.htmlに追加）

const params = new URLSearchParams(window.location.search);
const utmKeys = ['utm_source', 'utm_medium', 'utm_campaign', 'utm_content', 'ref'];
const utmData = {};
utmKeys.forEach(key => {
  const val = params.get(key);
  if (val) utmData[key] = val;
});
if (Object.keys(utmData).length > 0) {
  localStorage.setItem('shadowcast_utm', JSON.stringify(utmData));
  window.history.replaceState({}, '', window.location.pathname);
}

7. 競合対策

差別化メッセージ

vs シャドテン（¥10,780/月）
「人間のコーチに月1万円。それとも、AIのコーチに月480円。22倍の価格差、あなたならどちらを選びますか？」
vs ELSA Speak / Speak（¥1,500〜4,000/月）
「アプリのインストール不要。ブラウザを開くだけ。しかも月480円。」

防御戦略

  • ブランド認知の先行獲得：「シャドーイング × AI」で検索した時に最初に出てくる存在になる
  • コミュニティ構築：Discord / LINEオープンチャットでユーザー同士の交流を促進
  • 切り替えコストの埋め込み：練習履歴・ストリーク・統計データを蓄積させる
  • 毎週新機能リリース：常に「先を行っている」状態を維持する
