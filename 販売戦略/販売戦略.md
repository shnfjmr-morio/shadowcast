## 2. コンテンツマーケティング戦略

### 共通原則

**コンテンツの軸は「教育 × 共感 × 製品デモ」の3本柱**。比率は 5:3:2 が目安。

### X（Twitter）戦略

**投稿頻度**: 1日2〜3投稿（朝7時・昼12時・夜21時）

**コンテンツカテゴリと具体例**:

#### 教育系（50%）— フォロワー獲得用
```
「TOEICリスニング満点の人が全員やってること、それはシャドーイング。

でも問題は"正しくできてるかわからない"こと。

録音して聞き返しても、素人には自分の発音のどこが悪いか判断できない。

だからプロに月1万円払う人がいる。

…でも本当にそれ、人間じゃないとダメ？」
```

#### 共感系（30%）— 拡散用
```
「シャドーイングの月額サービス調べたら月10,780円だった。

年間約13万円。

同じことAIにやらせたら月480円。

年間5,760円。

差額12万円。

12万あったら実際に海外行けるんだよな。」
```

#### 製品デモ系（20%）— 転換用
```
「ShadowCastで30秒練習してみた。

1. 音声を聞く
2. 声に出して追いかける
3. AIが即座にフィードバック

ブラウザだけで完結。インストールなし。
しかも基本無料。

[デモ動画/GIF]」
```

**バズ狙い施策**:
- 「シャドテンに月1万払ってた人に聞きたい」系のアンケートツイート
- 著名な英語学習者へのリプライ営業（有益な情報を返す形で）
- 「#英語学習」「#TOEIC」「#シャドーイング」を活用（1投稿2個まで）

### YouTubeショート戦略

**投稿頻度**: 週3〜5本

**フォーマットパターン**:

1. **Before/After型**（最も効果的）
   - 構成：練習前の音声 → 2週間後の音声 → 「ShadowCastで毎日5分やっただけ」
   - 尺：30〜45秒

2. **比較型**
   - 構成：「月10,780円のサービス vs 月480円のShadowCast」
   - 尺：45〜60秒

3. **TIPSメ型**
   - 構成：「シャドーイングで絶対やってはいけない3つのこと」→最後にShadowCastをチラ見せ
   - 尺：30〜45秒

### TikTok戦略

**投稿頻度**: 週5〜7本（Xの素材を縦動画化して転用）

**TikTok特有の戦術**:
- 冒頭1秒で「えっ」と思わせるフック
- コメント欄で質問に丁寧に回答（コメント返し動画も作る）
- トレンドサウンドを活用（英語学習と自然に結びつくものだけ）

**全プラットフォーム共通の鉄則**:
- 動画内・投稿内でURLを直接貼らない。「プロフィールのリンクから」と誘導
- コメントへの返信は全件、最初の1時間以内に行う

## 4. 流入元トラッキング方法

### 4-1. UTMパラメータ設計

```
https://shadowcast.app/?utm_source={プラットフォーム}&utm_medium={メディア種別}&utm_campaign={キャンペーン名}&utm_content={コンテンツ識別子}
```

| チャネル | URL |
|----------|-----|
| Xプロフィール | `?utm_source=x&utm_medium=social&utm_campaign=profile` |
| X個別投稿 | `?utm_source=x&utm_medium=social&utm_campaign=launch&utm_content=comparison-tweet-0401` |
| YouTubeショート | `?utm_source=youtube&utm_medium=video&utm_campaign=shorts` |
| TikTokプロフィール | `?utm_source=tiktok&utm_medium=social&utm_campaign=profile` |
| Product Hunt | `?utm_source=producthunt&utm_medium=listing&utm_campaign=launch` |
| アフィリエイト | `?utm_source=affiliate&utm_medium=referral&utm_campaign=aff-{affiliate_id}` |
| 紹介リンク | `?ref={user_id}` |

### 4-2. 実装方法（index.htmlに追加予定）

```javascript
// ランディング時にUTMパラメータを保存
const params = new URLSearchParams(window.location.search);
const utmKeys = ['utm_source', 'utm_medium', 'utm_campaign', 'utm_content', 'ref'];
const utmData = {};
utmKeys.forEach(key => {
  const val = params.get(key);
  if (val) utmData[key] = val;
});
if (Object.keys(utmData).length > 0) {
  localStorage.setItem('shadowcast_utm', JSON.stringify(utmData));
  window.history.replaceState({}, '', window.location.pathname);
}
```

### 4-3. 計測ツール構成

1. **Cloudflare Web Analytics**（無料）: 全体トラフィック・リファラーの俯瞰
2. **カスタムUTMログ**: 流入元別の登録数・CVRを自前で記録
3. **週次レポート**: スプレッドシートで流入元別のCVRを集計（初期はこれで十分）

## 5. 海外展開戦略

### 5-1. 展開タイミングの判断基準（3条件すべて満たしたら開始）

1. 日本で有料転換率 5%以上が2週間継続
2. 致命的バグの報告が週0件
3. 月次チャーンレート（解約率）が10%以下

### 5-2. ターゲット市場の優先順位

| 順位 | 市場 | 理由 |
|------|------|------|
| 1 | 韓国 | 英語学習熱が日本と同レベル。TOEIC文化圏 |
| 2 | 台湾 | 英語学習ニーズ高。親日で日本発サービスへの心理的障壁低 |
| 3 | ブラジル・中南米 | 英語学習人口が急増中。安価なサービスへの需要大 |
| 4 | 英語圏（非ネイティブ向け） | 移民・留学生向け。市場規模大だがリーチコスト高 |

### 5-3. ローカライゼーション

- UI翻訳: 韓国語・繁体中国語・ポルトガル語・スペイン語
- 各市場のSNSアカウントを個別に開設
- 現地の英語学習系インフルエンサーに無料PRO + アフィリエイト報酬で提携

## 7. 競合対策

### 7-1. 模倣された場合の防御戦略

1. **ブランド認知の先行獲得**: 「シャドーイング × AI」で検索した時に最初に出てくる存在になる
2. **ユーザーコミュニティの構築**: Discord / LINEオープンチャットでユーザー同士の交流を促進
3. **切り替えコストの埋め込み**: 練習履歴・統計データ・連続記録（ストリーク）を蓄積させる
4. **新機能を毎週リリース**: 常に「先を行っている」状態を維持

### 7-2. 競合との差別化メッセージ

**vs シャドテン（¥10,780/月）**:
> 「人間のコーチに月1万円。それとも、AIのコーチに月480円。22倍の価格差、あなたならどちらを選びますか？」

**vs ELSA Speak / Speak（¥1,500〜4,000/月）**:
> 「アプリのインストール不要。ブラウザを開くだけ。しかも月480円。」

## まとめ：最重要ポイント3つ

1. **BETAで「使い続ける人がいる」ことを確認してからローンチする** — PMFがない状態でマーケティングしても穴の空いたバケツに水を注ぐだけ
2. **価格差を最大の武器にする** — 「シャドテンの22分の1」という比較は、どんなコピーライティングより強い
3. **スピードが唯一の堀** — 先にブランドを確立し、先にコミュニティを作り、先に海外展開する。「最初に知られた者が勝つ」
